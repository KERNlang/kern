# @kernlang/native

Kern React Native transpiler

Part of the [KERN monorepo](https://github.com/KERNlang/kern).

## Install

```bash
npm install @kernlang/native
```

## Usage

```ts
import { transpile } from '@kernlang/native';

const reactNativeCode = transpile(kernSource);
```
